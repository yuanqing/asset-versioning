Usage: asset-versioning replace [globs] [options]

Globs:
  One or more globs for matching files to perform a
  find-and-replace of original filenames with their versioned
  filenames.

Options:
  -b, --baseDirectory BASE_DIRECTORY  Directory in which to
                                      match files. Defaults to
                                      the current directory.
  -m, --manifestFile MANIFEST_FILE    A JSON file mapping the
                                      original filenames to the
                                      versioned filenames. Read
                                      from stdin if this is not
                                      specified.
  -h, --help                          Print this message.

Example:
  asset-versioning replace 'build/**/*' --manifestFile manifest.json
